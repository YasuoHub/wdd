# 问当地 (WenDangDi)

基于地理位置的即时互帮互助微信小程序 - MVP版本

## 🎯 项目简介

问当地是一款基于地理位置的互帮互助平台，连接有信息需求的求助者与对应地点的帮助者，通过积分激励机制，实现异地实时信息快速获取。

**核心价值**:
- 📍 **求助者**: 快速获取异地实时信息
- 💰 **帮助者**: 利用碎片时间赚取积分
- 🏆 **平台**: 构建互帮互助的社区生态

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0
- 微信开发者工具 >= 1.05
- CloudBase CLI >= 1.0

### 1. 克隆项目

```bash
git clone https://github.com/yourusername/wendangdi.git
cd wendangdi
```

### 2. 安装依赖

```bash
# 安装 CloudBase CLI
npm install -g @cloudbase/cli

# 登录腾讯云
tcb login
```

### 3. 初始化数据库

```bash
node scripts/init-db.js
```

### 4. 部署云函数

```bash
# 一键部署（推荐）
./scripts/deploy.sh

# 或手动部署
# 在微信开发者工具中，右键点击 cloudfunctions/ 目录下的函数
# → 创建并部署：云端安装依赖
```

### 5. 配置小程序

1. 用**微信开发者工具**打开 `miniprogram` 目录
2. 设置 AppID: `wx37902a802fff342d`
3. 点击**云开发**按钮，开通云开发环境
4. 修改 `cloudfunctions/config/index.js` 中的管理员OpenID

```javascript
admin: {
  openid: 'your-openid-here',  // 修改这里
  // ...
}
```

### 6. 配置模板消息

在微信小程序后台申请订阅消息模板：

1. 任务匹配通知
2. 积分到账通知  
3. 签到提醒

将模板ID填入 `cloudfunctions/config/index.js` 中。

### 7. 运行项目

点击微信开发者工具的**编译**按钮，即可在模拟器中预览。

## 🏗️ 技术架构

### 前端
- **框架**: 原生微信小程序
- **样式**: WXSS + 大字体模式适配
- **状态管理**: 本地存储 + 全局数据

### 后端
- **平台**: 腾讯云 CloudBase
- **云函数**: Node.js (19个函数)
- **数据库**: MongoDB (10个集合)
- **存储**: 云存储（图片等文件）
- **实时通信**: 数据库实时推送

### 核心算法
- **匹配算法**: 距离 + 类型 + 评分综合排序
- **积分系统**: 多重获取途径 + 冻结机制
- **防刷机制**: 设备/IP/频率多层限制

## 📝 配置文件

### 积分规则配置

编辑 `cloudfunctions/config/index.js`:

```javascript
points: {
  NEW_USER: 100,           // 新用户积分
  COMPLETE_PROFILE: 30,    // 完善信息奖励
  SIGN_IN: {
    BASE: 5,              // 基础签到积分
    STREAK_7: 10,         // 7天连续
    STREAK_30: 20         // 30天连续
  },
  INVITE: 50,             // 邀请奖励
  MIN_PUBLISH: 10,        // 最低发布积分
  BONUS_RATE: 0.05        // 好评奖励比例
}
```

### 业务规则配置

```javascript
business: {
  AUTO_SETTLEMENT_HOURS: 24,  // 自动结算时间
  MAX_MATCH_NOTIFY: 3,        // 最多通知帮助者数
  MAX_MESSAGE_LENGTH: 500,    // 消息最大长度
  MAX_IMAGES_PER_UPLOAD: 3    // 最多上传图片数
}
```
